# Triple M Studios — Developer Portfolio

Welcome to the official portfolio site of **Triple M Studios**, a personal showcase of software development projects, tools, and experiments by a passionate computer science graduate and aspiring developer.

## 🌐 Website

> [https://TripleMStudio.dev](https://TripleMStudio.dev)

This site is a collection of:

- 💻 **Web apps** (like a lyrics search tool using FastAPI and syncedlyrics)
- 📊 Data-driven projects (school funding, tutoring analytics, etc.)
- 🛠️ Tools built for productivity, automation, and education

## 🚀 Projects Featured

### 🔍 Lyric Search App

A web client built with FastAPI + syncedlyrics that fetches time-tagged lyrics using open APIs.

- Strips or keeps timestamps
- Mobile-friendly design
- Built with vanilla JS, FastAPI backend, and CSS

### 📂 Auto Backup Tool *(in progress)*

A Python watchdog-based script that monitors development folders and syncs to a flash drive with portable Git for mobility.

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **Backend:** FastAPI (Python)
- **Tools:** Git, Render, Watchdog, SyncedLyrics, Python 3.12+

## 📦 Deployment

Hosted on:

- Frontend: GitHub Pages/Domain
- Backend: Render (FastAPI)
